Цей проєкт є вебплатформою для екологічного моніторингу, створеною в межах лабораторної роботи з дисципліни «Екологічний моніторинг». Основна мета системи полягає в автоматизації розрахунків екологічних індексів та оцінки ризиків для здоров'я населення, що виникають через забруднення повітря та води. Платформа дозволяє користувачам вводити дані про рівень забруднення та отримувати оцінки щодо потенційного ризику для здоров'я. Ці оцінки базуються на методиках, визначених Міністерством охорони здоров’я України, що дозволяє отримати точні та науково обґрунтовані результати.

Проєкт побудований на повному стеку MERN (MongoDB, Express, React, Node.js), що забезпечує ефективну взаємодію між клієнтською частиною, сервером і базою даних. Зокрема, для створення інтерфейсу використовувався React, що дозволяє розробити динамічні форми для введення даних і надання користувачам зручного інтерфейсу для перегляду результатів обчислень. Веб-застосунок має інтуїтивно зрозумілий інтерфейс, що робить процес введення даних і отримання результатів швидким і простим для користувачів.

Система дозволяє автоматизувати складні математичні розрахунки, що використовуються для визначення неканцерогенного (HQ) та канцерогенного (CR) ризиків для здоров’я населення. Ці розрахунки здійснюються згідно з методиками МОЗ України, що забезпечує відповідність міжнародним стандартам в екологічному моніторингу. Введені користувачем дані перевіряються на відповідність екологічним нормативам, що дозволяє отримати точні результати і уникнути помилок через некоректні дані. Така валідація забезпечує додаткову надійність системи.

Веб-платформа також дозволяє зберігати історію розрахунків і переглядати попередні результати, що особливо корисно для подальшого аналізу змін у рівнях забруднення. Це допомагає моніторити екологічну ситуацію в часі та коригувати стратегії щодо захисту здоров’я населення. Дані зберігаються в базі даних MongoDB, що є високопродуктивною і гнучкою NoSQL базою, і дозволяє ефективно працювати з великими обсягами даних.

Проєкт реалізовано із використанням серверної частини на Node.js з фреймворком Express. Цей стек дозволяє створити швидкий і надійний сервер для обробки запитів від користувачів і здійснення необхідних обчислень. Сервер взаємодіє з MongoDB для збереження результатів і надає API для обміну даними з фронтендом.

Однією з особливостей цього проєкту є інтеграція штучного інтелекту для оптимізації структури коду та логіки обробки даних. Для цього використовувались інструменти на базі великих мовних моделей (Gemini та DeepSeek), які допомогли в оптимізації алгоритмів і перевірці правильності розрахунків. Використання AI дозволило значно прискорити процес розробки, покращити точність виконуваних операцій та забезпечити коректність результатів.

Проєкт складається з трьох основних компонентів: фронтенд частина реалізована за допомогою React, бекенд побудований на Node.js з Express, а для збереження результатів використовуються моделі MongoDB. Користувачі можуть вводити дані через зручний інтерфейс, отримувати розрахунки, а також зберігати історію своїх введених результатів. Кроки для запуску проекту включають клонування репозиторію, встановлення залежностей, налаштування підключення до MongoDB і запуск серверної та клієнтської частини, що дозволяє швидко налаштувати і запустити систему на локальному сервері.

Завдяки використанню технологій, таких як MERN та інтеграція AI для оптимізації коду, проєкт є ефективним і масштабованим рішенням для екологічного моніторингу та оцінки ризиків для здоров'я. Вебплатформа може бути використана не тільки для моніторингу поточної ситуації з забрудненням, але і для аналізу тенденцій зміни екологічних показників на основі збережених даних.
